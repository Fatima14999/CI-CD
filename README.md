# CI/CD Final Project

This project demonstrates a complete CI/CD pipeline using:

- GitHub Actions
- OpenShift Pipelines
- Tekton Tasks
- ESLint
- Jest Testing
- Buildah Deployment

## Features

- Linting with ESLint
- Unit testing with Jest
- Container image build with Buildah
- Deployment using OpenShift Pipelines

## Pipeline Steps

1. cleanup
2. git-clone
3. eslint
4. jest
5. buildah
6. deploy
